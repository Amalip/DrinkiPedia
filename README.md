# DrinkiPedia

#Requirements to run
- Android Studio version 4.0.2
- Gradle version 6.1.1
- Min SDK version 26
- Kotlin version 1.4.10
- Java version 8


#Steps to run project

- Open AndroidStudio
- Select "Get from Version Control"
- Paste "https://github.com/Amalip/DrinkiPedia.git" on URL
- Click on "Clone"
- Open project
