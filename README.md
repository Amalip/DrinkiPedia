# DrinkiPedia

#Requirements to run
- Android Studio version 4.0.2
- Gradle version 6.1.1
- Min SDK version 26
- Kotlin version 1.4.10
- Java version 8


#Steps to run project

1 - Open AndroidStudio
2 - Select "Get from Version Control"
3 - Paste "https://github.com/Amalip/DrinkiPedia.git" on URL
4 - Click on "Clone"
5 - Open project
